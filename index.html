Созданные веб-приложения!

Twitch Follow Info - сайт, где я использовал api стриминговой платформы Twitch, для получения данных и авторизации. Для создания сайта использовались технологии react, redux toolkit, typescript. Также использовались некоторые сторонние библиотеки:
  • react-simple-oauth2 - для авторизации с помощью твича ouath2
  • react dom router - для машрутизации сайта
  • react-content-loader - для создании скелетона, когда данные ещё не получены из API
Сайт позволяет увидеть активные каналы и информацию о канале(название стрима, кол-во зрителей и т.д.), на кототорые зафоловлен пользователь, также можно увидеть все подписки пользователя.

Помодоро сайт - созданный для себя. Я пользуюсь данной методикой для формировании своего рабочего дня. Сайт был создан на реакте, с использованием разных хуков и сторонних компонентов. Планирую дальше дополнять этот сайт для себя, так как не все идеи ещё реализовал (поэтому некоторые кнопки ещё не функциональны). Сайт был сверстан по своему дизайну.

CloudBudget - адаптивный одностраничный сайт сделанный на html и css. Сайт был сверстан по макету из фигмы.

Sway - адаптивный одностраничный сайт, где использовался чистый js для реализации dropdown кнопок, бургер меню и гибкой адаптивности шапки. Также использовался api сайта openstreetmap для реализации карты на сайте. Сайт был сверстан по макету из фигмы.

Hobby Garden - адаптивный одностраничный сайт, с использованием html, css и js для реализации бургер меню и гибкой адаптивности. Сайт был сверстан по макету из фигмы.
